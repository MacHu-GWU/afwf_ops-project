Install
------------------------------------------------------------------------------

``afwf_ops`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install afwf-ops

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade afwf-ops
